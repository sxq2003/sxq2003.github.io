# Nunjucks Error: xxxx unexpected token: }}

写 SSTI 之类的文章时会遇到这样的报错，是因为 hexo 也有渲染引擎

解决方式：在不想被渲染的文章的 Front-matter 添加一行 disableNunjucks: true

# 特殊格式被自动解析
